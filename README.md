Minimal RPG demo
=================
This game so far is a small demo created entirely by copilot. What follows is all it. I'll make a more complicated game once I learn more about how to make web games.


This repository contains a tiny browser RPG demo to help test your static site and GitHub Pages setup.

How to run
- Open `index.html` in a browser (double-click or serve with a static server).
- The demo is client-side only (no build step required).

What it includes
- `index.html` — page and UI container.
- `style.css` — simple styles.
- `demo.js` — a compact, single-file RPG with movement, random encounters, and simple battles.

Notes
- This is intentionally small and self-contained so you can quickly verify your site is serving JS and assets correctly.
